Variables, Constantes y Valores {{bloque.title}} {{bloque.subtitle}} {{bloque.descrip}} Ejemplos var mi_nombre: string = "Ignacio";
var cualquier_cosa: any = "Ignacio";
let edad: number = 31;
const PESO_MAX: number = 50; También podemos declararlas así: var mi_nombre = "Ignacio";
let edad = 31;
const PESO_MAX = 50;

En este caso el lenguaje va a definir el tipo de dato segun el valor que almacenamos inicialmente y, según el lenguaje en el que estemos, no nos dejará cambiar su valor por un tipo de dato diferente. Por ejemplo: no nos va a dejar guardar un número en mi_nombre. Aún así, hay lenguajes que sí o sí te piden definir el tipo de dato antes de asignar un valor.
